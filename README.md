📌 Project Overview

This is a Django REST Framework based backend project for managing product sales, carts, and authentication using JWT.

🚀 Features
     User Authentication (JWT)
     Product Management (CRUD)
     Cart Management
     REST API endpoints
🛠️ Tech Stack
     Python
     Django
     Django REST Framework
     PostgreSQL
📂 Project Structure
     authentication/ → Handles user login & JWT
     products/ → Product CRUD operations
     productsales/ → Project settings
     manage.py → Django entry point
⚙️ Setup Instructions
     git clone https://github.com/Arun10-git/MyproductSale.git
     cd MyproductSale
     python -m venv .venv
    .venv\Scripts\activate
     pip install -r requirements.txt
     python manage.py runserver
📡 API Endpoints
    /product/ → Get all products
    /product/<id>/ → Get product by ID
    /carts/ → Cart operations
